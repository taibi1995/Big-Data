# Big Data — Analysis and Processing

Project focused on processing and analyzing large datasets with PySpark and performant Python tooling.

Highlights
- PySpark examples for data ingestion and transformations
- Reproducible notebooks for exploratory analysis
- Guidance for scaling locally to cluster

Installation
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Run
```bash
jupyter notebook "projet bigdata.ipynb"
```
